# Calendary_arq_web

Proyecto web de calendario desarrollado con arquitectura separada en **backend** y **frontend**.  
El sistema utiliza **MongoDB** como base de datos, ejecutada mediante **Docker**, y un backend en **Node.js + TypeScript**.

## Estructura del proyecto

```bash
proyecto calendario/
│
├── calendar-backend/
│   ├── src/
│   ├── docker-compose.yml
│   ├── package.json
│   └── .env
│
└── calendar-frontend/
    ├── src/
    ├── package.json
    └── ...
